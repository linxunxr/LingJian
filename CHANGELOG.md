# 更新日志

本文件记录「灵鉴」各版本的变更内容。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，按版本倒序排列。

发布时，CI 会自动从本文件提取对应版本段落，写入 `latest.json` 的 `notes` 字段，作为客户端更新提示中展示的更新日志。

---

## v0.1.5 - 2026-07-03

### 修复
- COS 关闭全球加速，改回普通地域域名（ap-guangzhou）
- COS 上传增强断点续传：分块重试 + 指数退避 + 碎片清理 + 进度可见
- 首页版本号显示修正

### 新增
- 启用 DevTools（F12 调试控制台），便于排查首页白屏等运行时问题

## v0.1.4 - 2026-07-03

### 修复
- 设置保存后首页仍提示配置不完整：apiKey 改存 settings.json 替代 keyring（部分 Windows 环境 keyring 写入静默失败）
- 应用图标显示为蓝色占位：用 tauri icon 重新生成全套图标，确保与源设计一致

### 优化
- COS 产物按版本分目录归档：安装包上传到 `/<版本>/` 子目录，`latest.json` 留根目录，避免所有版本混在一起
- COS 改用全球加速域名（CI 上传 + 客户端更新 + 用户下载均走加速接入点）

## v0.1.0 - 2026-06-09

首个可用版本：Path of Idle Immortals 日志分析工具。

### 新增
- 日志分析核心：通过 SCF 端点拉取日志、解析分类、本地 SQLite 落库与缓存
- 图表可视化：基于 Chart.js 的时间线图表与错误聚合面板
- 虚拟滚动列表，支持海量日志条目流畅浏览
- 问题列表：自动拉取 Issue、状态切换、点选分析
- Issue 操作：关闭/重开、评论、标签菜单（经 SCF 代理 GitHub）
- Markdown / JSON / CSV 多格式导出
- 凭证加密存储（系统钥匙串 keyring）
- 设置页连接验证与首次启动引导
- 键盘快捷键（Ctrl+F 搜索 / Esc 返回）
- 数据目录便携存储：跟随安装位置、可自定义、自动迁移
- 应用自动更新（Tauri Updater，腾讯云 COS 分发）

### 修复
- 下载日志后解析失败：兼容裸数组与包裹对象两种格式，补全 category 字段

### 优化
- 统一前后端类型契约
- 引导流程简化为两步（SCF URL + API Key）
- 移除 GitHub Token 凭证管理与直连逻辑，统一经 SCF 端点代理
